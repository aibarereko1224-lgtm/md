# Moon Dust

A minimalist, immersive media tracking app for movies, TV shows, and books.

## Features

- Immersive background scenes (Forest, Starry Night, Stream, Desert)
- Glassmorphism UI with adaptive colors
- Search and add media with poster selection
- Intelligent library grouping by month/season
- Particle mask for reviews
- Export images for sharing

## Setup

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## API

Uses TMDB API for movie/TV search. Get an API key from [TMDB](https://www.themoviedb.org/) and add to environment variables.
